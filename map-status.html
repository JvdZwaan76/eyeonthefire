About This Fire Map

This map displays near real-time active fire data from NASA's Fire Information for Resource Management System (FIRMS).

The data comes from the VIIRS and MODIS instruments aboard NASA's satellites, which detect thermal anomalies that indicate active fires.

Data is refreshed automatically every 5 minutes.
